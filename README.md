# Lab 11 – Simulation with SimPy

## Purpose
This lab focuses on:
- modeling systems over time
- using SimPy processes
- simulating interactions between events

Detailed instructions are provided in Canvas.

---

## Requirements

Your program must include:
- a traffic light process
- a car process
- a car arrival process

---

## Simulation

- print output showing system behavior
